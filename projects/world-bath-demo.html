World Bath

Relax. Explore. Connect.

A small visual-novel/gacha-style demo is practical for this repository because it can be built as one static HTML page: no backend, no accounts, no real purchases, and assets loaded from GitHub Pages.

1. Location menu + scene carousel

Select a location, cycle through themed backgrounds, click the active character, and read a short visual-novel dialogue beat.

background concept

Feasibility

Yes, this is possible. The current project is a static GitHub Pages-style site, so the demo should use plain HTML, CSS, and JavaScript first. It keeps hosting simple and avoids build tooling.

  • Best MVP: 3 locations × 3 backgrounds × 3 characters × 3 skins.
  • Image count: 1 key art + 9 backgrounds + 3 base transparent characters + 9 skin portraits = 22 images.
  • Formats: AVIF/WebP for backgrounds, PNG/WebP with transparency for characters.
  • Safety: adult characters only, tasteful spa/fashion styling, no nudity.

2. Character viewer + skin carousel

Each companion can have profile data, story hooks, favorite gifts, and a carousel of collectible outfits/skins.

    collectible look

    3. Recommended generation prompts

    Generate in consistent batches. Keep the same art direction, camera language, character reference, age statement, and negative prompt across all images.

    Location background prompt

    Wide cinematic background for a tasteful gacha visual novel, [COUNTRY] traditional bathing destination, [SEASON/TIME], no people, warm lantern light, detailed architecture, steam, water reflections, layered depth, 16:9, game background, painterly high detail, safe for work

    Character base prompt

    Full-body transparent-background character sprite, adult woman age 24+, [COUNTRY] companion, elegant spa travel outfit inspired by local textiles, friendly expression, visual novel character art, consistent face, clean silhouette, hands visible, safe for work, PNG alpha

    Skin / outfit prompt

    Portrait/card art of the same adult character, [SKIN THEME] collectible outfit, tasteful fashion, matching accessories, gacha rarity frame space, confident pose, no nudity, high-detail illustration, 3:4 character card

    Negative prompt

    minor, childlike, nude, explicit, see-through clothing, distorted hands, extra fingers, bad anatomy, duplicate face, text artifacts, watermark, logo, low resolution, blurry

    Demo uses CSS placeholder art so the team can validate flow before spending image-generation budget.
